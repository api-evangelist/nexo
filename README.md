# Nexo (nexo)

Nexo is a crypto-backed lending and earning platform offering a REST API for managing crypto credit lines, earning interest on digital assets, swapping assets, and portfolio management. The Nexo Pro API provides programmatic access to advanced trading (spot, futures, TWAP), account balances, order management, and trade history via REST and WebSocket interfaces.
